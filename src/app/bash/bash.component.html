Introduction

Bash is a Unix shell and command language written by Brian Fox for the GNU Project as a free software replacement for the Bourne shell. First released in 1989, it has been used widely as the default login shell for most Linux distributions. It has been widely ported to various operating systems, including macOS, and is the default shell on most Android devices.
